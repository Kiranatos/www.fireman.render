Инструкция к применению

План здания должен быть в формате растрового изображения (например, jpeg, bmp, png, gif и другие). Добавление фонового рисунка осуществляется нажатием на иконку "Добавить план здания". После нажатия на данную иконку откроется стандартный диалог открытия файлов, где и осуществляется выбор файла изображения. После выбора файла рабочая область программы может выглядеть следующим образом:

РИСУНОК

Рисование схемы эвакуации осуществляется с помощью панели быстрого запуска "Палитра элементов", которая изображена на рисунке.

РИСУНОК

Сначала вы должны добавить начало пути эвакуации. Для этого нажмите на иконку РИСУНОК, далее переместите курсор мыши на место рабочей области, откуда начинается, по вашему мнению, эвакуация, и кликните левой кнопкой мыши (на рабочей области появится пиктограмма начала эвакуации). Таким образом можно добавлять начало каждого из потоков эвакуации. На рисунке показан пример рабочей области с размещенными на ней началами потоков.

РИСУНОК

Подобным образом действуйте и для добавления окончания пути эвакуации, но сначала нажмите на иконку РИСУНОК"Выход". Помните, что множество входов может быть соединено только с одним выходом.

После того как вы создали входы и выходы, необходимо их соединить отдельными участками. Для этого используются узлы участков. Нажмите на иконку для добавления узлов. Размещайте узлы в тех местах, где будут соединены отдельные участки пути эвакуации.

Начало и окончание потоков эвакуации, а также узлы вы можете нумеровать по собственному усмотрению, для этого в параметрах соответствующего элемента (чтобы вызвать окно параметров, щелкните два раза левой кнопкой мыши) необходимо изменить значение поля "Узел №".

Теперь настало время соединить все узлы, входы и выходы между собой. Для этого щелкните на иконке "Участок", далее нажмите левой кнопкой мыши на один из элементов схемы (при этом за курсором будет тянуться пунктирная линия), а затем на другой элемент схемы (или на пустое пространство рабочего поля). В результате этого между элементами появится зеленая линия, изображающая отдельный участок.

Обратите внимание: при отсутствии информации о длине участка программа будет отображать на участке знак вопроса. Отображение знака вопроса возможно отменить в окне настроек параметров программы.
